
# Contributing

## Tests

```bash
pytest
```

## Documentation

The documentation is created with [great-docs](https://posit-dev.github.io/great-docs/). To generate and render the site:

```bash
great-docs build
```

To open in your browser:

```bash
great-docs preview
```

## All-contributors

To add names or contributions to the README, first install the [All-Contributors CLI tool](https://allcontributors.org/cli/installation/):

```bash
npm i -D all-contributors-cli
```

Then run:

```
npx all-contributors
```
